# Product Management - Backend (Spring Boot)

This is the backend part of the Product Management system, developed using Spring Boot.  
It provides REST APIs to perform CRUD operations on products.

## 🚀 Technologies Used:
- Java
- Spring Boot
- Spring MVC
- Spring Data JPA
- Oracle SQL Database
